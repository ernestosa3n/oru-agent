## üß≠ Main Screens

### 1. Dashboard
- Displays real-time insights generated by the deployed intelligence agent.
- Insight types: Risk, Trend, Opportunity, Anomaly.
- First cards are fully interactive and open the detail view.

### 2. Agent Configuration (Wizard)
A 3-step guided flow to create an intelligence agent:

1. **Data Sources**  
   HTS, HCS, smart contracts, network metrics.
2. **Objectives**  
   Monitoring, risk detection, opportunity discovery, trend analysis.
3. **Frequency**  
   Real-time, minute, hour, daily.

Includes a final ‚ÄúDeploy Agent‚Äù confirmation.

### 3. Learning Log
- Shows memory entries and reasoning steps.
- Demonstrates compounding intelligence behavior.

### 4. Settings
- Network mode
- Theme switch
- Agent status
- System preferences

## üéØ Purpose of the Prototype

The prototype is designed to:
- Demonstrate how on-chain intelligence agents can be created and deployed.
- Show the user experience of interacting with ORU.
- Present a realistic product vision for the Hedera AI & Agents hackathon.

It is not intended to be fully functional but provides a complete, professional interface flow.
